ВАЖНО: Для работы сайта нужно открыть ДВА отдельных окна терминала.

=== ТЕРМИНАЛ 1: БЭКЕНД (Сервер Python) ===
1. Откройте терминал в папке проекта (g:\vkr_project\).
2. Введите команды:
cd backend
python -m pip install -r requirements.txt
python -m uvicorn main:app --reload

(Если сервер запустился успешно, вы увидите надпись "Application startup complete".)

=== ТЕРМИНАЛ 2: ФРОНТЕНД (Сайт React) ===
1. Откройте ВТОРОЙ терминал в папке проекта (g:\vkr_project\).
2. Введите команды:
cd frontend
npm.cmd install
npm.cmd run dev

3. После запуска вы увидите ссылку, например: http://localhost:5173 
4. Откройте эту ссылку в браузере.

ПРИМЕЧАНИЕ:
- Если вы уже установили зависимости (pip install / npm install), то каждый раз их вводить не нужно.
- Достаточно просто запускать:
  Backend: python -m uvicorn main:app --reload
  Frontend: npm.cmd run dev
