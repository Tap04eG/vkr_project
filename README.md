# KidsLearn - Образовательная Платформа 🚀

Веб-приложение для обучения детей (6-8 лет) с элементами геймификации. Платформа объединяет учеников, учителей и родителей для эффективного и интересного процесса обучения.

## 🛠 Технологический Стек

*   **Frontend:** React, Vite, React Router, CSS модули.
*   **Backend:** Python, FastAPI, SQLAlchemy.
*   **База данных:** SQLite (можно переключить на PostgreSQL).
*   **Аутентификация:** JWT (JSON Web Tokens).

## ✨ Возможности

### 🎓 Ученик
*   **Личный кабинет:** Просмотр уровня и заработанных очков (XP).
*   **Задания:** Выполнение заданий от учителя.
*   **Геймификация:** Получение опыта за выполнение задач, повышение уровня.

### 🍎 Учитель
*   **Управление классами:** Создание классов и добавление учеников.
*   **Выдача заданий:** Назначение задач конкретным ученикам с указанием награды (XP).
*   **Мониторинг:** Просмотр списка учеников и их успеваемости.

### 🏠 Родитель
*   **Привязка ребенка:** Подключение аккаунта ребенка по логину.
*   **Статистика:** Просмотр прогресса ребенка (уровень, XP, выполненные задачи).

## 🚀 Установка и Запуск

### Предварительные требования
*   Node.js (v18+)
*   Python (v3.9+)

### 1. Настройка Бэкенда (Backend)

1.  Перейдите в папку backend:
    ```bash
    cd backend
    ```
2.  Создайте виртуальное окружение:
    ```bash
    python -m venv .venv
    ```
3.  Активируйте окружение:
    *   **Windows:** `.venv\Scripts\activate`
    *   **Mac/Linux:** `source .venv/bin/activate`
4.  Установите зависимости:
    ```bash
    pip install "fastapi[all]" sqlalchemy pydantic passlib[bcrypt] python-jose[cryptography] python-multipart
    ```
5.  Запустите сервер:
    ```bash
    uvicorn main:app --reload
    ```
    Сервер будет доступен по адресу: `http://127.0.0.1:8000`
    Документация API (Swagger): `http://127.0.0.1:8000/docs`

### 2. Настройка Фронтенда (Frontend)

1.  Откройте **новый** терминал и перейдите в папку frontend:
    ```bash
    cd frontend
    ```
2.  Установите зависимости:
    ```bash
    npm install
    ```
    *Также потребуются иконки и axios:*
    ```bash
    npm install lucide-react axios react-router-dom
    ```
3.  Запустите приложение:
    ```bash
    npm run dev
    ```
    Приложение откроется по адресу: `http://localhost:5173`
